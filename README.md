# Kimchi Premium Tracker
## Instructions

Set the following env variables
- `EXCHANGE_RATES_PATH`
- `PREMIUM_PATH`
- `COINBASE_PATH`
- `COINONE_PATH`
- `NOTIFY_PATH`
- `NOTIFY_LOW`
- `NOTIFY_HIGH`
- `DISCORD_SERVER`
- `DISCORD_HOOK`
- `FIXER_API_KEYS`

Configure run.sh and run-rates.sh on a cronjob
